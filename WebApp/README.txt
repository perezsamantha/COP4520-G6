HOW TO RUN ASYC SERVER SIMULATION
1. Install Eclipse IDE for Java EE Developers https://www.eclipse.org/downloads/
2. Install Apache Tomcat https://tomcat.apache.org/download-80.cgi
3. Create a Dynamic Web Project or import the existing project
4. Select Run as -> Run On Server
5. Go to http://localhost:8080/COP4520/asyncSimulator
